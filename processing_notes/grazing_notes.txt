Data from the A&M GAN lab are found in:
"\\ARS-DATA\Archive\HarmelExit\riesel\GrazinglandQuality\data\GANS reports"

The data was considered to have two classes:
Near-Infrared Spectroscopy (NIRS) Data
Nutrition Balance (NutBal)

The data files are .pdf. Three python pdf reader packages were found and 
tested: pypdf, PyMuPDF (fitz), and pdfplumber. pdfplumber was chosen 
because of it's ability to extract tables and text from complex layouts.
Unfortunately the extract_tables function did not work well. The function
extract_text worked well in converting the pages into text stored as 
it's own variable. The text was then scrubbed and identifiers to split
the data was implented. The data was appended to the corresponding list
and once the file has been read those lists were inserted into pandas
dataframes. These dataframes will be written to .csv files for future
ease.

Exploring more files in the root folder it was found that there is very
little consistency in the formatting of the data within the .pdf. 
These inconsitencies can be different placement of data within the form,
data spanning multiple rows (being different from file to file), data
can be present in one file while missing in another. Variable name 
differences.
This has proven to be extremely difficult to find a way to automate
the reading and collection of the data.
